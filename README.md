### Installation:
```
pip3 install --upgrade git+https://github.com/gReen7aRRow/python-project-52.git
```

**Main commands:**
- Installing dependencies - ***make install***.
- Testing - ***make test***.
- Installing migrations - ***make migrate***.
- Launch the application - ***make run***.
